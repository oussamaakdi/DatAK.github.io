Ressources

These materials have helped me learn and grow, and I believe they can be equally valuable to you.

Books

  • Introduction to Machine Learning with Python : A comprehensive guide to machine learning concepts and their practical applications using Python.
  • Pattern Recognition and Machine Learning by Christopher M. Bishop: This book provides in-depth insights into pattern recognition and machine learning algorithms.
  • Beginning DAX with Power BI: Discover the world of Data Analysis Expressions (DAX) in Power BI for advanced data modeling.

Documentation

  • scikit-learn User Guide: A detailed user guide for scikit-learn, a powerful machine learning library in Python.
  • TensorFlow Documentation: Explore the comprehensive documentation for TensorFlow, an open-source machine learning framework.

Tools and Software

  • Jupyter Notebook: An open-source web application that allows you to create and share documents that contain live code, equations, visualizations, and narrative text, making it an excellent choice for data exploration and analysis.
  • PyTorch: An open-source machine learning library developed by Facebook's AI Research lab, particularly known for its flexibility and dynamic computation graph.
  • Kaggle: A popular platform for data science competitions, datasets, and collaboration. It offers a wide range of datasets, notebooks, and competitions for data enthusiasts.
  • Azure Machine Learning: A cloud-based machine learning service provided by Microsoft Azure. It enables you to build, train, and deploy machine learning models using Azure's cloud infrastructure.
  • Power BI: A business analytics service by Microsoft that provides interactive data visualization and business intelligence capabilities. It's widely used for creating data reports and dashboards.
